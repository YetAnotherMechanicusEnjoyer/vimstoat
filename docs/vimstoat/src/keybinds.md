# Keybinds

## Basic Navigation

| Key | Action     |
| --- | ---------- |
| `h` | Move left  |
| `j` | Move down  |
| `k` | Move up    |
| `l` | Move right |

## Mode Switching

| Key   | Action                                                           |
| ----- | ---------------------------------------------------------------- |
| `i`   | Enter Insert mode to the left of the block cursor                |
| `I`   | Enter Insert mode, at the start of the line                      |
| `a`   | Enter Insert mode to the right of the block cursor               |
| `A`   | Enter Insert mode, at the end of the line                        |
| `o`   | Enter Insert mode, and create a new line below the current line. |
| `O`   | Enter Insert mode, and create a new line above the current line. |
| `:`   | Enter Command mode                                               |
| `/`   | Search (enters Command mode with `/` prefix)                     |
| `v`   | Enter Visual mode (future)                                       |
| `Esc` | Enter Normal mode                                                |

## Fuzzy Finder

| Key  | Action       |
| ---- | ------------ |
| `fs` | Find server  |
| `fc` | Find channel |
| `ff` | Find friend  |
| `fm` | Find message |
